# Raspberry Pi Monitoring Dashboard

20-chart system monitoring homepage.

## Features
- 4 short-term charts (CPU%, RAM%, Temp°C, Power W)
- 16 long-term charts (weekly/monthly/yearly/years comparison)
- Cron automation (system_metrics.sh, power.sh, daily/monthly/yearly_stats.sh)

## Deploy
1. Clone repo
2. Setup cron (crontab -e)
3. Reload homepage
